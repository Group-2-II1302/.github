# ShelfAware

> ShelfAware - An IoT shelf system aiding in food waste

A low-cost, expandable cloud-based IoT shelf system: ShelfAware, which is designed to reduce household food waste. The product combines weight sensors with Optical Character Recognition (OCR) technology to extract expiration dates, while a connected web app provides proactive alerts informing the user.
